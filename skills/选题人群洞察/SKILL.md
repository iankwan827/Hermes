# 选题人群洞察

你的任务是：拿到一个选题，倒推出这个选题背后在跟谁对话，提炼人群洞察，然后重构出这个人群真正需要被回答的问题，最后用多个思维模型拆解底层逻辑。

**核心逻辑**：选题是表层，人群是载体，真正的问题藏在人群的处境里。

---

## 三步流程

### Step 1：人群还原

拿到选题，先不急着分析，做一件事：**把这个选题背后的那个人还原出来。**

输出格式：

```
## 这条选题在跟谁说话

**她是谁**
{具体描述，不是标签。不是"年轻人"，是"刚入职两年、每个月发完工资剩两千块、知道靠工资不行但不知道怎么动的那批人"}

**她现在的处境**
{她的现实状况：钱、时间、资源、认知卡点}

**她已经知道什么**
{她不是白纸，她已经有哪些认知}

**她卡在哪**
{她不是不想，是卡在什么地方没法动}

**她真正想要的是什么**
{不是她说的那个，是她真正渴望的那个}
```

---

### Step 2：重构问题

基于这个人群，**把原始选题重构成一个更精准的底层问题。**

重构原则：
- 往往是反转或倒推：选题说"如何做 X"→ 真正的问题可能是"那些已经做到的人，他们绕开了什么"
- 具体化数字：模糊的"赚钱"变成具体的"30万/100万"
- 去掉她认为自己需要的（方法），找到她真正需要的（认知转变）

输出格式：

```
## 重构后的底层问题

原始选题：{xxx}
重构问题：{重构后的提问，更精准、更有穿透力}

重构逻辑：{为什么这样重构，一句话说清楚}
```

---

### Step 3：多思维模型分析

用 4-5 个不同的思维模型，从不同角度分析重构问题背后的共性底层逻辑。

**必须覆盖的思维模型维度**（从中选最相关的 4-5 个）：

| 模型 | 适用场景 |
|------|---------|
| 杠杆原理 | 时间/精力/资金的放大器在哪里 |
| 稀缺性思维 | 什么是真正的稀缺资源 |
| 竞争优势 | 人群如何建立不对称的优势 |
| 价值创造 vs 价值传递 | 哪个环节才是真正的价值来源 |
| 系统思维 | 她卡住的地方是不是系统性的障碍 |
| 逆向思维 | 成功案例做对了什么，失败案例做错了什么 |
| 机会成本 | 她目前的选择是在用什么换什么 |
| 第一性原理 | 去掉所有假设，最底层的逻辑是什么 |
| 网络效应 | 有没有可以滚雪球的机制 |

输出格式：

```
## 底层逻辑拆解

### 思维模型一：{模型名称}
{从这个模型角度，这个问题的底层逻辑是什么}
→ 对内容创作的启发：{具体可以用的观点/角度}

### 思维模型二：{模型名称}
{分析}
→ 对内容创作的启发：{具体可以用的观点/角度}

（依此类推 4-5 个模型）

## 共性底层逻辑

跨越所有模型，这些分析指向同一个东西：
{1-2 句话提炼，这就是内容要说的核心认知}
```

---

## 完整输出结构

```
# 选题人群洞察：{原始选题}

## 这条选题在跟谁说话
（Step 1 内容）

## 重构后的底层问题
（Step 2 内容）

## 底层逻辑拆解
（Step 3 内容）

## 共性底层逻辑
（最终提炼）

```

## 使用示例

**输入**：如何打开赚第一桶金的能力

**Step 1 输出**：
> 这条选题在跟谁说话：靠打工的人。她通过出卖自己的时间和能力换取工资，知道这条路的上限，但还没找到用自己的能力直接和市场交换的方式。她有能力，但没有把能力变成产品的方法。

**Step 2 输出**：
> 重构问题：普通人如何不靠打工，赚到 30 万/100 万？——绕开她认为自己需要的"方法"，直接问成功的人做对了什么。

**Step 3**：用"价值创造 vs 价值传递""杠杆原理""竞争优势"等模型分析，发现共性：**赚到第一桶金的人都找到了一个"可以被复制的价值单元"——不是出卖时间，而是打包了可以重复交付的东西。**

---

## 注意事项

- 人群还原必须**具体到场景**，不能停留在人群标签（"年轻女性"不是人群，是标签）
- 重构问题要**有点反常识**，太直接的重构没价值
- 思维模型分析的最后必须**收敛到一个共性逻辑**，不能留着 4 个互不相关的分析
- 最后的"下一步"是帮用户把洞察转化成内容骨架的桥梁，不要跳过
