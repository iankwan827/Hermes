# 阶段 1：吃透选题

**灵魂问**：你真的懂这个东西为什么存在吗？

**输出**：选题的底层情绪 + 赛道定位 + 90% 痛点清单 + 数据支撑

---

## 执行流程

### Step 1：接收选题

用户输入选题后，先确认：
- 这是一个产品名？（如：掌机王 SP、iPhone 16）
- 这是一个话题？（如：为什么李亚鹏赚不到钱）
- 这是一个现象？（如：年轻人不愿意上班）
- 这是一个问题？（如：如何在 6 个月内赚到第一桶金）

### Step 2：可选 - 调用选题顾问

如果用户加了 `--evaluate` 参数，或者你判断这个选题可能有致命伤，先调用 `/选题顾问` 评估：

```
现在调用 /选题顾问 评估这个选题是否值得做...
```

如果选题顾问返回"✗ 不推荐"且命中致命伤，直接终止流程，输出：

```
## 评估结果：这个选题不适合做视频

[选题顾问的致命伤分析]

[素材回收建议]

---

如果你仍然想继续，可以输入"强制继续"，但建议先调整选题。
```

如果用户说"强制继续"，才进入 Step 3。

### Step 3：双层研究员预设

**Layer 1：产品拆解**

读取 `templates/researcher-tech.md`，用这个预设分析选题，输出 9 栏知识卡。

**Layer 2：情绪穿透**

读取 `templates/researcher-emotion.md`，基于 Layer 1 的结果，往下挖一层，输出 5 栏情绪卡。

### Step 4：合并输出双层知识卡

```
# 选题：{选题名称}

## Layer 1：产品拆解

[9 栏知识卡内容]

---

## Layer 2：情绪穿透

[5 栏情绪卡内容]

---

## 核心洞察

基于以上分析，这个选题的核心是：
- **表层**：[从 Layer 1 提取，这是什么产品/技术]
- **里子**：[从 Layer 2 提取，它触到人的哪一块]
- **钩子来源**：[从 Layer 2 第 5 栏提取，用户失去它会失去什么]

这个洞察会在阶段 2 用来设计开头。
```

### Step 5：保存 checkpoint

自动保存到：
```
~/.claude/projects/-Users-liujia/memory/video-workflow-state/{timestamp}-phase1-knowledge-card.md
```

### Step 6：Gate - 等待用户确认

```
---

【Gate 1】请确认：

这张双层知识卡对「{选题}」的理解是否准确？

- 如果准确，回复 **"过"** 或 **"继续"**，我会进入阶段 2
- 如果有偏差，请告诉我哪里需要调整

（这一步很重要：如果理解错了，后面 7 步全在放大错误）
```

用户回复"过"或"继续"后，才进入阶段 2。

---

## 注意事项

1. **双层预设必须都跑**：不能只跑 Layer 1，Layer 2 是找钩子的关键
2. **Layer 2 第 5 栏是命门**：这一栏挖不到东西，说明选题没有情绪抓手
3. **Gate 不能跳过**：即使用户催促，也要等确认
4. **如果用户说"不对"**：回到 Step 3 重新分析，不要进入阶段 2

## 输出示例

### 示例：掌机王 SP

**Layer 1：产品拆解**

1. **一句话讲明白**：90 年代最火的游戏攻略杂志，教你怎么通关红白机游戏
2. **解决了什么问题**：那个年代没有互联网，卡关了没人教，这本杂志就是攻略百科
3. **背后的硬件/软件/算法**：纸质杂志 + 编辑部实际通关后写的攻略 + 读者投稿
4. **普通用户能感受到什么**：拿到杂志 = 拿到通关秘籍，在同学面前有话语权
5. **品牌为什么强调它**：那个年代的游戏玩家都看过，是一代人的集体记忆
6. **可能被误解的地方**：现在的人会觉得"不就是一本杂志吗"，不理解信息稀缺时代的价值
7. **适合短视频的 3 个选题**：
   - 一本杂志，如何成为 90 后的童年圣经
   - 为什么掌机王 SP 能卖 30 年
   - 那些年，我们为了一本掌机王 SP 做过的事
8. **适合直播间讲的 3 句话**：
   - 90 后应该都知道，小时候攒一个月零花钱就为了买这本杂志
   - 那个年代没有百度，卡关了只能靠这本杂志
   - 拥有这本杂志 = 在同学中是游戏大神
9. **5 个关键词**：攻略、通关、童年、稀缺、话语权

**Layer 2：情绪穿透**

1. **替代了什么关系**：替代了"没人陪我玩游戏"的孤独感，替代了"我不知道怎么通关"的无助感
2. **激活了哪层情绪**：童年时第一次靠自己通关游戏的成就感，在同学面前炫耀的优越感
3. **身体层面在做什么**：蹲在被窝里打手电筒看，听到爸妈脚步声就藏起来；在学校传阅，几个人围着一本杂志讨论
4. **为什么封神**：90 年代游戏攻略信息极度稀缺，拥有这本杂志 = 掌握稀缺资源 = 群体身份标记
5. **失去它会失去什么**：会重新感到孤单（没有东西陪伴），会重新失去在同学面前的话语权（不再是游戏大神）

**核心洞察**

- **表层**：一本 90 年代的游戏攻略杂志
- **里子**：它是信息稀缺时代的"权威代偿"，给了孩子们掌控感和社交货币
- **钩子来源**：用户失去它，会失去童年时那种"我掌握了别人不知道的秘密"的优越感

## 常见问题

**Q：如果选题不是科技产品怎么办？**
A：Layer 1 的预设仍然适用，只是"硬件/软件/算法"那一栏改成"背后的机制/内容/形式是什么"

**Q：如果 Layer 2 第 5 栏挖不到东西怎么办？**
A：说明这个选题缺乏情绪抓手，建议回到选题顾问重新评估，或者调整选题角度

**Q：用户说"不对"怎么办？**
A：问清楚哪里不对，重新跑 Step 3，不要进入阶段 2

**Q：用户说"跳过 Gate"怎么办？**
A：不能跳过。告诉用户："这一步是整个流程的地基，如果理解错了，后面 7 步全在放大错误。请确认一下，只需要回复'过'就可以继续。"
