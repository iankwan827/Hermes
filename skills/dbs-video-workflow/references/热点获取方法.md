# 抖音热搜获取方法

## 方法：OpenCLI Chrome浏览器 + eval提取

当curl API全部失效时（2026年常见），用OpenCLI复用Chrome登录态直接访问抖音热搜页面。

### 步骤

```bash
# 1. 打开抖音热搜页面
opencli browser chrome open "https://www.douyin.com/hot"
# 返回 page ID（如 D4B2D2E82951495C53E67398C2B3A8D4）

# 2. 等待页面加载
sleep 3

# 3. 提取页面文本内容
opencli browser chrome eval "document.body.innerText" --tab <PAGE_ID>

# 4. 提取热搜榜（带热度的条目）
# 热搜榜数据在页面文本的后半部分，格式为：rank\n标题\nX.X万热度
opencli browser chrome eval "
(function() {
    let allText = document.body.innerText;
    let lines = allText.split('\\n');
    let result = [];
    for (let i = 0; i < lines.length; i++) {
        if (lines[i].match(/\\d+\\.\\d+万热度/)) {
            let rank = lines[i-2] || '';
            let title = lines[i-1] || '';
            result.push(rank + '. ' + title + ' | ' + lines[i]);
        }
    }
    return result.slice(0, 30).join('\\n');
})()
" --tab <PAGE_ID>
```

### 输出格式

```
1. 坚持中孟友好大方向不动摇 | 1211.0万热度
2. 高考志愿填报有哪些误区 | 1187.4万热度
3. 我国设立养老服务师职业资格制度 | 1171.3万热度
...
```

### 注意事项

- **page ID需要从open命令的返回值中获取**，每次打开都会生成新的page ID
- **eval中的变量名不要与页面已有变量冲突**（用IIFE包裹避免 `Identifier has already been declared` 错误）
- **OpenCLI用的是Chrome浏览器**（不是Camofox），需要Chrome已启动并安装Browser Bridge扩展
- **如果Chrome未运行**：先启动Chrome（`terminal background=true`），等待10-15秒让Browser Bridge连接，用 `opencli daemon status` 验证
- **页面文本很长**：先用 `document.body.innerText.substring(0, 3000)` 试探结构，再提取完整内容

### 失败的API（2026-06-26验证全部不可用）

以下curl API已全部失效：
- `https://www.douyin.com/aweme/v1/web/hot/search/list/` → 返回空
- `https://www.iesdouyin.com/web/api/v2/hotsearch/billboard/aweme/` → Access Denied
- `https://weibo.com/ajax/side/hotSearch` → 失败
- `https://weibo.com/ajax/statuses/hot_band` → 失败
- 各种第三方热搜API（gumengya、vvhan、tenapi等）→ 全部失败

**结论**：不要尝试curl API获取热搜，直接用OpenCLI浏览器方案。
