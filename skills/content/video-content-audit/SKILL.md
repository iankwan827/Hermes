## 触觉条件
用户写完视频文案/标题后，要求审核。

**⚠️ 自动触发（2026-06-09）：**
Agent写完文案后，必须在呈现给用户之前自动跑一遍审核，不要等用户提醒。用户原话："你没跑skill啊"——审核是内建流程，不是可选步骤。审核未通过不交付。

**⚠️ 事实审核范围：** 不仅审核文案结构，还要审核事实准确性。用户原话："必须点进热搜看内容再推荐，不能只看标题就推"。任何引用的热点事件（如浪姐几公、具体票数），必须搜索确认后才能写入文案。不确定的事实宁可模糊化处理（"最近有个姐姐"），也不能写错具体细节。

## 审核流程

### 1. 读取最新经验教训

读取发展日志（D:/hermes-agent/douyin-skill/references/发展日志.md）中的"经验教训"部分，获取从数据中总结的规则。

### 2. 逐条审核

#### A. 标题审核（致命级）

| 检查项 | 规则 | 级别 |
|--------|------|------|
| 悬念钩子 | 财经/热搜类标题必须有悬念（「你被骗了」「这才是真相」「99%的人都不知道」），不能平铺直叙 | ❌致命 |
| 认知冲突 | 标题要制造"我以为A其实B"的认知冲突 | ❌致命 |
| 字数 | 15-20字，不能太长也不能太短 | ⚠️重要 |
| 标签 | ≤5个，不能堆砌 | ⚠️重要 |

#### B. 开头Hook审核（致命级）

| 检查项 | 规则 | 级别 |
|--------|------|------|
| 前2秒 | 必须有冲突/悬念/反常识，不能平铺直叙 | ❌致命 |
| 认知冲突 | "你以为X？其实Y"结构 | ❌致命 |
| 脱稿提示 | 用（）内嵌正文对应位置，不单独放表格 | ⚠️重要 |

#### C. 文案结构审核（重要级）

| 检查项 | 规则 | 级别 |
|--------|------|------|
| **字数/时长审核** | **写完必须数中文字数。粤语口播正常语速约3-4字/秒，90秒上限=270-360字。超过360字必须精简后再提交审核。这是最常见的审核漏洞。** | ❌致命 |

| 检查项 | 规则 | 级别 |
|--------|------|------|
| 阿强叙事 | 用阿强做叙事载体：铺垫→反转→吐槽→引出主题 | ⚠️重要 |
| 段子/钩子 | **每30-45秒必须有一个留人钩子，不能有超过40秒的"平铺区"** | ❌致命 |
| 钩子密度实操 | **写完后标注每个钩子的秒数位置，检查相邻钩子间隔是否>40秒。如果有，必须补钩子** | ❌致命 |
| 时长 | 1.5分钟以内（90秒） | ❌致命 |
| 互动引导 | 结尾必须有反问观众的互动句 | ⚠️重要 |
| 逻辑延续 | 续集必须承接上一集故事线 | ⚠️重要 |
| 结构性回声 | 开头的关键词/数字应在正文中间再次呼应（如"99%"前后呼应） | ⚠️重要 |
| **对话不重复** | **正文只保留一段完整对话/案例，不要同一件事换不同角度反复讲。如果前面有阿强对话，后面不要再展开类似对话** | ❌致命 |
| **排比句精简** | **排比句（如"早上穿什么——X买的。中午吃什么——X吃什么"）如果删掉不影响关键信息，必须删掉** | ⚠️重要 |
| **第三人称检查** | **文案中"他们""一个人"等第三方表述，检查是否应改为"我们"增强代入感** | ⚠️重要 |

#### D. 标签审核（重要级）

| 检查项 | 规则 | 级别 |
|--------|------|------|
| 热搜话题标签 | 如果选题来自热搜，必须把热搜原话题作为第一个标签（吃热搜流量） | ❌致命 |
| 垂类标签 | 后面跟账号垂类标签（栋笃笑/粤语吐槽等，吃垂类流量） | ⚠️重要 |
| 实际使用 | 标签必须搜实际在用的（微博热搜/抖音垂类），不能自己编 | ⚠️重要 |
| 数量 | ≤5个 | ⚠️重要 |
| 相关性 | 标签必须与内容直接相关 | ⚠️重要 |
| **身份匹配** | **标签必须匹配自己的内容身份（栋笃笑/吐槽号），不能照搬源素材的标签（除非确实匹配）** | ⚠️重要 |

**标签公式**：热搜原话题标签 + 精准人群标签 + 账号垂类标签（2个）+ 泛流量标签
例：#金价暴跌不是经济问题 #黄金 #栋笃笑 #粤语吐槽 #打工人

**⚠️ 标签选择不要过度纠结（2026-06-20 用户纠正）：** 标签确定后直接用，不要反复修改、不要逐条解释、不要给太多选项。用户原话："随便把……"——说明已经不耐烦了。正确做法：搜到标签后直接确认，不要过度分析。标签不是文案，不需要打磨到完美。

#### E. 事实审核（致命级）

| 检查项 | 规则 | 级别 |
|--------|------|------|
| 不确定事实 | 不确定的事实要确认或删换素材 | ❌致命 |
| 粤语金句 | 不要直接写，用剧情/对话体现 | ⚠️重要 |
| 时间线 | 文案时间线必须符合现实逻辑 | ❌致命 |
| 热点轮次 | 综艺/赛事的轮次必须搜索确认（如浪姐几公），不能凭印象写 | ❌致命 |
