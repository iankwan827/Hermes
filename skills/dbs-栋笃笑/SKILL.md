# dbs-栋笃笑工作流

两步串联：dbs写初稿 → 黄子华风格改终稿

---

## 第一步：dbs-video-workflow

按原版4阶段执行：
1. 阶段1：吃透选题（双层知识卡）
2. 阶段2：钩住人群（调用 /开头 skill）
3. 阶段3：承诺蜕变（A→B路径）
4. 阶段4：聊透输出（初稿）

**输出**：初稿文案

---

## 第二步：黄子华风格改写

加载 `huang-zihua-perspective` skill，用以下框架改写初稿：

### 栋笃笑叙事结构

```
爆款开头（1-2句，定话题）
    ↓
人物故事铺垫（"我同阿强讲……"）
    ↓
现实对照
    ↓
炫耀→反转
    ↓
吐槽人物 → 引出观点
    ↓
金句收尾
    ↓
反问互动（跟故事呼应）
```

### 改写要点

1. **开头**：用黄子华模板库（45个模板）改写，制造反常识/悬念
2. **阿强叙事**：铺垫→反转→吐槽，用对话体制造共鸣
3. **金句收尾**：一句话点睛，让人回味
4. **节奏**：每30-45秒一个钩子，反转一句话打完
5. **粤语**：用地道粤语词，不用普通话词

### 黄子华模板速查

| 模板 | 句式 |
|------|------|
| 反常识 | 很多人说XX，其实一点都不 |
| 两种人 | 这个世界上只有两种人，一是XX，二是YY |
| 被迷死了 | 我刚刚被XX迷死了 |
| 不看好 | 一开始没有人看好XX，偏偏他最争气 |
| 别再XX了 | 别再XX了，你的YY会当真 |
| 我突然领悟了 | 我突然领悟了，XX背后的逻辑和动机 |
| 我发现 | 我发现我们对XX的认知，存在一个极其YY的标准 |
| 分享一个醍醐灌顶 | 分享一个醍醐灌顶吧，我们大部分人都是XX |
| 差点气死 | 我真的差点气死了，分享X个XX |
| 今天说透 | 今天一个视频把XX说透 |

---

## 审核

终稿完成后跑 `video-content-audit`：
- 字数：粤语4字/秒，90秒上限=360字
- 开头：必须有爆款钩子
- 阿强：铺垫→反转→吐槽
- 钩子密度：每30-45秒一个
- 金句：要有
- 互动：结尾反问

---

## 注意事项

- dbs阶段正常走，不用管风格
- 黄子华阶段只改风格，不改内容逻辑
- 用户改完就是终版，不要加回删掉的内容
- 用户打比方是教角度，不是素材，不要写进文案
