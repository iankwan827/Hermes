# 跨 Skill 组合模式

## 黄子华 Skill + dbs-video-workflow

### 组合逻辑
- dbs-video-workflow 提供**结构**（选题→开头→蜕变→终稿）
- 黄子华 Skill 提供**灵魂**（自嘲、荒诞、粤语、金句）

### 执行顺序
```
1. dbs-video-workflow 阶段1 → 理解选题（痛点+情绪+人群）
2. dbs-video-workflow 阶段2 → 设计开头（用黄子华自嘲风格改写）
3. dbs-video-workflow 阶段3 → A→B蜕变路径
4. 黄子华 Skill → 全文改写（加入粤语表达、荒诞对比、层层递进）
5. dbs-video-workflow 阶段4 → 脱稿框架+终稿
```

### 黄子华风格检查清单
- [ ] 开头有自嘲吗？
- [ ] 有荒诞对比吗（常识vs现实）？
- [ ] 话题是自然滑动的吗（不是分段）？
- [ ] 有"但系"做转折连接吗？
- [ ] 有金句收尾吗？
- [ ] 用了粤语特有表达吗（乜嘢/㗎/嗰阵）？
- [ ] 是挑战权威而不是嘲笑弱者吗？

### 示例：六一儿童节文案
- 选题：大人骗小孩
- 开头：自嘲"考第一有奖励"
- 递进：圣诞老人→爱情→我也开始骗人
- 升华：被骗也是爱
- 金句：圣诞老人系马云
