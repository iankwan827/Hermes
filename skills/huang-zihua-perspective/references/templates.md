# 粤语句式模板库

> 16个验证过的模板，每个含核心技巧、黄子华原文案例、分析、套用方式。

---

## 1. 荒诞平行 ⭐⭐⭐⭐⭐

用"同样"把两个不相关的事叠在一起，荒诞感自动产生。

句式：同样系XX，一个YYY，一个ZZZ。同样，你同你同事，一个YYY，一个ZZZ。

原文案例：
```
同样系主角，一个7年仲喺地上跑，一个2年就住上天。
同样，你同你同事，一个做7年都系打杂，一个听日就升职做主管。
```

---

## 2. 掰手指数（数数递进）⭐⭐⭐⭐

用数数制造"越来越多→越来越荒谬"的节奏感。

句式：XX跑完跑YY，YY跑完跑ZZ……跑完N个，结果呢？依然XXX。

原文案例：
```
蒙德跑完跑璃月，璃月跑完跑稻妻，稻妻跑完跑须弥，
须弥跑完跑枫丹，枫丹跑完跑纳塔……
跑完6个国家，至冬嘅门都未入到。
```

---

## 3. 否定式反常识 ⭐⭐⭐⭐

先说大众认知，一句否定，再给真相。

句式：你以为XXX？其实YYY。

---

## 4. 同一个XX，差距大到离谱 ⭐⭐⭐⭐

强调"同一起点/同一来源"，让差距更荒谬。

句式：同一个XX，一个YYY，一个ZZZ。你话边个先系亲生嘅？

---

## 5. 反问式互动收尾 ⭐⭐⭐

把话题从"他们"拉回"你"，制造代入感。

句式：你试过XXX未？评论区讲下。

---

## 6. 阿强叙事体 ⭐⭐⭐⭐⭐

用"我同阿强讲"引出人物，用阿强的蠢/不信/倒霉制造笑点。

句式：我同阿强讲：[观点/事实]。阿强话：[蠢反应/不信/吐槽]。

要点：阿强的反应=观众的嘴替。阿强不是知识来源，是共鸣载体。

---

## 7. 荒诞解构童话/常识 ⭐⭐⭐⭐⭐

拿一个大家公认的事实，用逻辑推翻，推到荒谬。

原文案例（白雪公主）：
```
白雪公主，几咁纯洁无瑕。但系谂真啲，
白雪公主识得嗰个系王子，
灰姑娘识嗰个又系王子，
钟意锡青蛙嘅女人锡咗只青蛙一啖，
只青蛙变出嚟嘅又系王子！
根本呢个王子都不知几花心，周围沟女。
仲无良到扮青蛙去沟添！
```

套用：换主体→找规律（3个以上案例有共同点）→推到荒谬

---

## 8. 金句反义翻转 ⭐⭐⭐⭐

把一个正能量金句翻转成讽刺。

原文案例：
```
做嘢做得咁辛苦为咩啊？买啊慰劳下自己啊！
```

套用：选正能量金句→放进揭示陷阱的语境→笑点来自观众以为你在说正能量，结果你在讽刺

---

## 9. 三部分结构（完整栋笃笑短片）⭐⭐⭐⭐⭐

三段式递进：A场景对比 → 荒诞平行 → 阿强故事收尾。

```
Part 1：A场景对比（游戏/影视/事件，用掰手指数或直接对比）
Part 2：荒诞平行（"同样"停顿，把A场景映射到打工人现实）
Part 3：阿强故事（对话体收尾，阿强的吐槽=观众嘴替）
```

---

## 10. 回水 Running Gag ⭐⭐⭐⭐⭐

把一个简单的词变成全场互动仪式，反复喊到变成笑点。

原文案例：
```
我听到有人根本唔係嗌任何内容，係纯粹嗌——回水！
回水！回水！回水！
我Michael Jackson都五十岁，佢问谁回水？
我点鬼知！
总之齐嗌回水！
```

套用：选一个简单有力的词→先给合理语境→扩散到荒谬对象→让观众一起喊

---

## 11. 历史/文学人物解构 ⭐⭐⭐⭐⭐

拿一个大家熟知的历史人物，用现代视角解构，找到荒谬点。

原文案例（孔明老婆）：
```
孔明老婆姓黄，外号叫做阿醜。
如果你係一件猪扒，同你猪扒到一个地步，
你个外号叫做猪扒，係两种境界。
一个人要醜成点，先至可以有个绰号叫阿醜呢？
```

套用：选熟知人物→找"不合理"的点→用现代语言重新解读→推到荒谬

---

## 12. 递进式崩溃表演 ⭐⭐⭐⭐

用重复+递进，把一个小事放大到崩溃级别。

原文案例（林黛玉"起惊悲痛"）：
```
你话畀林妹妹中咗六合彩：
"吓？我中咗六合彩？好呀！"（起）
即刻谂："咁多人听到我中六合彩..."（惊）
跟住谂："我中咗六合彩咪好囉，点解要有两注重呢..."（悲）
最后谂："吓？我身体咁差，就算中咗安慰奖我都洗唔曬..."（痛）
起、惊、悲、痛——林妹妹冇得开心㗎。
```

套用：选一个应该正面的事→设计3-4个心理递进阶段→每个阶段消解前一个的正面情绪

---

## 13. 观众互动陷阱 ⭐⭐⭐⭐

先问观众问题，再用答案制造笑点。

原文案例：
```
有边位朋友顶唔顺粗口嘅？拍下手啦。
有几位拍下手。
呢啲人个心理真係好虚弱下...
有边位朋友顶唔顺？我奉劝你——
（连续30个"唔係"制造焦虑感）
```

套用：问一个让部分观众举手的问题→用举手结果制造下一个笑点→让没举手的人也紧张

---

## 14. 梵高理财式荒诞类比 ⭐⭐⭐⭐

用一个"不应该"的组合，推到逻辑结论。

原文案例：
```
有间银行推出梵高理财——梵高帮你理财。
梵高係世界上鼎鼎大名嘅大话家。
冇有人比他更穷。
梵高係有精神病㗎——一个有精神病嘅人帮你理财，
然后你有金融风暴，合情合理呀！
```

套用：选一个"不应该"的组合→逐条列举不匹配特质→最后一句推到荒谬结论

---

## 15. 粤语粗口节奏控制 ⭐⭐⭐⭐

用粗口的"预感"制造紧张，用延迟释放笑点。

原文案例：
```
佢话：我嘟你咩呀！
佢当然唔係用个嘟字啦。
边个想听下原文呀？
想听原文嘅朋友可以去搭小巴。
畀几蚊就有粗口听啦，点解要畀几百蚊入场听？
```

套用：用替代词暗示粗口→假装不知道观众在想什么→最后用"廉价化"消解紧张

---

## 16. 夫妻相处之道（递进式吐槽）⭐⭐⭐⭐

用真实夫妻对话，递进到荒谬。

原文案例：
```
你问百分之九十九嘅老夫老妻，相處之道係乜嘢？
都係俾呢个答案：忍呀忍呀忍呀，好快就過㗎啦。
咩要過咗人生？咁忍你做咩？

有時你會瞓唔著，而家你好明呀——因為佢瞓得著呀嘛。
我咁多嘢忍，佢都瞓得著。

忍到最終狀態：忍咗等於冇忍，冇忍等於忍咗。
```

套用：选一个"大家都说靠X"的共识→用真实细节放大X的荒谬→推到哲学化结论
