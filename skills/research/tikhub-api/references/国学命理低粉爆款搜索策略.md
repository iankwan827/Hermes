## 扩充到1000+的策略

### 最佳方案：作者主页挖掘（需TikHub余额）

搜索接口有去重限制（172个关键词只搜出481条），要突破1000条必须换思路：

1. 用搜索接口拿到80-100条低粉爆款
2. 用 `fetch_one_video` 获取每条视频作者的 `sec_uid`
   - 响应结构：`data.aweme_detail.author.sec_uid`（注意不是 `data.author`）
3. 用 `fetch_user_post_videos` 扒每个作者主页所有视频
   - 需要付费余额，免费额度用完返回402
4. 从主页视频中筛选低粉爆款
5. 每个作者可贡献几十条新视频，50个作者轻松破千

脚本：`D:/hermes-agent/scripts/tikhub_expand_authors.js`

### 其他方案

1. **搜长尾问句**：如"八字看婚姻怎么"、"什么时候遇到正缘"、"伤官见官会怎样"
2. **搜标签组合**：如"国学+搞笑"、"命理+吐槽"、"易经+动画"
3. **搜相邻领域**：养生+中医+禅修+冥想+修行+心灵成长
